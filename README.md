- ChatBot para WhatsApp em Node.js


Este é um ChatBot desenvolvido em Node.js, utilizando a biblioteca Baileys para se conectar ao WhatsApp Web.
O objetivo do projeto é oferecer uma base simples, organizada e humana para criação de bots de atendimento, automações e assistentes personalizados para WhatsApp.


- Funcionalidades


-Responde mensagens automaticamente
-Sistema de comandos por palavras-chave
-Estrutura modular e fácil de entender
-Suporte para integração com APIs externas
-Mantém sessão ativa para uso contínuo
-Respostas humanas e naturais


- Tecnologias Utilizadas


Node.js
Baileys
Axios
Dotenv
Nodemon (opcional para desenvolvimento)
